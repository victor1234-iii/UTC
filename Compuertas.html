Sistema de Compuertas Lógicas con Control de Relé

Introducción

Este proyecto consiste en la implementación de un sistema basado en compuertas lógicas que permite controlar un relé para manejar dispositivos eléctricos de mayor potencia utilizando seńales de baja energía.

Objetivos

  • Diseńar un sistema funcional basado en compuertas lógicas
  • Comprender el uso del relé en sistemas electrónicos
  • Aplicar lógica binaria en circuitos reales

Funcionamiento

El sistema trabaja con seńales binarias (1 y 0). Dependiendo de las combinaciones de entrada, las compuertas lógicas activan o desactivan el relé, permitiendo controlar el paso de corriente.

Compuertas Utilizadas

  • Compuerta AND
  • Compuerta OR
  • Compuerta NOT

Materiales

  • Protoboard
  • Cables jumper
  • Resistencias
  • Relé
  • LEDs
  • Transistor
  • Circuitos integrados (74LS08, 74LS32, 74LS04)
  • Fuente de alimentación

Conclusión

El proyecto demuestra la aplicación de compuertas lógicas en sistemas reales, permitiendo automatizar procesos y controlar dispositivos de forma eficiente y segura.
