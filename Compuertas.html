📌 Introducción

Este proyecto consiste en el desarrollo de un sistema basado en compuertas lógicas que permite controlar un relé para manejar dispositivos eléctricos de mayor potencia utilizando señales de baja energía.

A través de este sistema se aplican conceptos de electrónica digital, permitiendo automatizar procesos de forma segura y eficiente.

🎯 Objetivo

Diseñar e implementar un sistema utilizando compuertas lógicas para controlar un circuito mediante un relé.

Objetivos específicos:

  • Comprender el funcionamiento de las compuertas lógicas
  • Aplicar lógica binaria en circuitos electrónicos
  • Construir un sistema funcional de control

📚 Marco teórico

Las compuertas lógicas son dispositivos electrónicos que operan con valores binarios (1 y 0), representando verdadero o falso.

Las principales compuertas utilizadas son:

AND: activa la salida si todas las entradas son verdaderas.
OR: activa la salida si al menos una entrada es verdadera.
NOT: invierte el valor de la entrada.

Estas compuertas permiten tomar decisiones dentro de un sistema electrónico.

🧰 Materiales

  • Protoboard
  • Cables jumper
  • Resistencias
  • Relé
  • LEDs
  • Transistor
  • CI 74LS08 (AND)
  • CI 74LS32 (OR)
  • CI 74LS04 (NOT)
  • Fuente de alimentación 5V

⚙️ Funcionamiento

El sistema recibe señales de entrada mediante interruptores, las cuales son procesadas por las compuertas lógicas.

Dependiendo de la combinación de entradas, el relé se activa o desactiva, permitiendo controlar el paso de corriente.

Esto permite encender o apagar dispositivos de manera automática.

✅ Conclusión

Este proyecto demuestra la aplicación de la lógica digital en sistemas reales, permitiendo controlar dispositivos eléctricos de manera eficiente mediante señales simples.
