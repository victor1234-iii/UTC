📌 ¿Qué es?

Una cerradura eléctrica en protoboard es un sistema que permite controlar el acceso mediante una contraseña o señal electrónica.

🎯 Objetivo

Diseñar e implementar una cerradura electrónica con Arduino.

📍 Propósito

Aplicar conocimientos de electrónica y programación.

🧰 Materiales

  • Protoboard
  • Arduino Uno
  • Pantalla LCD 16x2
  • Teclado matricial 4x4
  • Servo motor
  • Cables jumper
  • Resistencia de 330Ω
  • Buzzer

✅ Conclusión

Este proyecto permite comprender el uso de sistemas electrónicos para el control de acceso, aplicando programación y electrónica básica.
