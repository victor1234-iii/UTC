📌 Introducción

Este proyecto consiste en un sistema automatizado para controlar las condiciones de un invernadero utilizando sensores y actuadores.

🧰 Componentes

  • Arduino (controlador principal)
  • Sensor DHT11 (temperatura y humedad)
  • Pantalla LCD 16x2
  • Relays
  • Bomba de agua
  • Ventilador
  • Foco

⚙️ Funcionamiento

Los sensores envían datos al Arduino, el cual analiza la información y decide qué acción realizar.

Dependiendo de los valores, activa los relays para encender o apagar los dispositivos.

🧠 Lógica del sistema

Si la humedad es baja → se activa la bomba

Si la humedad es alta → se activa el ventilador

Si la temperatura es baja → se enciende el foco

✅ Conclusión

Este sistema permite automatizar el control ambiental de un invernadero, mejorando la eficiencia y reduciendo la intervención humana.
